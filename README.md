# Dados Climáticos de Piracicaba / Climate Data of Piracicaba

## 🇧🇷 Português

Este exercício usa como base de dados uma única tabela com dados meteorológicos diários coletados desde o ano de 1902 pelo Departamento de Engenharia de Biossistemas, da Escola Superior de Agricultura “Luiz de Queiroz” - Universidade de São Paulo (LEB/ESALQ/USP).

À série de dados original foram acrescidos os seguintes campos:
* **`Estiagem`**: variável que categoriza o dia por grupo de “dias seguidos de seca”.
* **`Semestre`**: categoriza o semestre (1 ou 2).
* **`ElNino`**: categoriza por intensidade do fenômeno El Niño.
* **`Estacao`**: indica a estação climática (Verão 22/12-20/3, Outono 21/3-20/6, Inverno 21/6-22/9, e Primavera 23/9-21/12).

A mais recente versão dessa tabela (no formato `.xlsx`) pode ser encontrada no [GitHub](INSIRA_O_LINK_AQUI), porém iremos analisar somente a estiagem.

## 🇺🇸 English

This exercise uses as its database a single table with daily meteorological data collected since 1902 by the Department of Biosystems Engineering at Escola Superior de Agricultura "Luiz de Queiroz" - Universidade de São Paulo (LEB/ESALQ/USP).

The following fields were added to the original data series:
* **`Estiagem`**: a variable that categorizes the day by a group of "consecutive dry days".
* **`Semestre`**: categorizes the semester (1 or 2).
* **`ElNino`**: categorizes by the intensity of the El Niño phenomenon.
* **`Estacao`**: indicates the weather season (Summer 22/12-20/3, Autumn 21/3-20/6, Winter 21/6-22/9, and Spring 23/9-21/12).

The most recent version of this table (in `.xlsx` format) can be found on [GitHub](INSIRA_O_LINK_AQUI), however, we will analyze only the dry spell.
